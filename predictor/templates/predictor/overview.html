⭐ Core Features of SkinSight

A complete AI-powered health support prototype.

Feature Overview

🔍 AI Skin Disease Detection

  • 🔸 Scan or upload a skin image
  • 🔸 Deep learning model analyzes image
  • 🔸 Identifies most likely condition from 23 classifications
  • 🔸 Provides prediction with confidence score

👤 Personal User Accounts

  • 🔸 Secure account creation
  • 🔸 Easy Login

📁 History & Progress Tracking

  • 🔸 Automatic storage of past scans
  • 🔸 Dedicated History page per user
  • 🔸 Monitor condition progression over time

📊 Health Insights & Smart Recommendations

  • 🔸 Detailed result page for each scan
  • 🔸 Indicates if medical attention is recommended
  • 🔸 Urgency level (monitor / soon / urgent)
  • 🔸 Highlights important red flags
  • 🔸 AI confidence score for transparency

🧾 Exportable Scan History (CSV)

  • 🔸 Download full scan history as CSV
  • 🔸 Offline access to records
  • 🔸 Shareable summary for consultation

📝 Exportable Pre-Diagnostic Report (CSV)

  • 🔸 Download your result as a CSV file.
  • 🔸 Can be used as a pre-diagonostic summary before visiting a dermatologist.
  • 🔸 Helps support more informed medical consultations.

Together, these features demonstrate a full AI-to-application pipeline — combining deep learning, backend engineering, and modern user experience design.
